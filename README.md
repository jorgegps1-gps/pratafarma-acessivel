# PrataFarma Acessível

Protótipo Front-End (HTML, CSS, JS) de e-commerce de **Medicamentos e Suplementos para Idosos**, com foco em **acessibilidade (WCAG)**.

## Páginas
- `index.html` – Home com catálogo, busca e controles de acessibilidade.
- `produto.html` – Detalhes do produto + alerta para itens com prescrição (ANVISA).
- `carrinho.html` – Itens, quantidades, frete estimado e totais.

## Acessibilidade
- HTML semântico, skip link, foco visível, alto contraste, A+/A-,
- Modo simplificado, aria-live, leitura em voz alta (Web Speech API).

## Como executar
Abra `index.html` no navegador. Não requer backend.

## Aviso
Este é um protótipo acadêmico; os dados são ilustrativos.
